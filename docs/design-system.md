 # Design System

## Design goals
- Modern and premium, but restrained rather than flashy
- Mobile-first and highly usable for volunteers, canvassers, and field organizers
- Professional enough for campaign managers and candidates
- Fast, calm, trustworthy, and practical under pressure
- Consistent with the login screen as the visual reference point

## Core visual principles
- Soft structure: rounded shells, soft borders, diffuse shadows
- Strong contrast where it matters: primary actions, page titles, field states
- Spacious layout: the interface should breathe
- Operational clarity: every screen should prioritize the next action
- Mobile-first: field workflows must work beautifully on a phone first

## Design tokens
- Backgrounds: warm white / off-white / pale slate
- Primary text: slate 900 / high-contrast dark neutral
- Secondary text: slate 500-600
- Primary accent: warm orange for calls to action and focus states
- Secondary accent: muted green for soft environmental depth only
- Radius (page shell): 32px equivalent
- Radius (cards, panels, modals): 20-24px equivalent
- Radius (buttons and inputs): 16-20px equivalent
- Spacing: 8px base scale
- Shadow: soft, diffuse, low-contrast

## Typography
- Sentence case for all interface language
- Bold and clean headings
- Highly legible body copy
- No tiny text on mobile

## Forms
- Inputs must be tall, touch-friendly, and clearly labeled above the field
- Strong orange focus states for active fields
- Validation errors obvious but calm
- Use segmented controls, chips, toggles, and large buttons to reduce typing

## Components
- Primary button: dark slate or warm orange fill, white text, large height, rounded corners
- Secondary button: white fill, light border, slate text
- Text input: tall field, labels above, 16-20px radius, strong orange focus state
- Metric card: white card, soft shadow, clear metric hierarchy
- Status chip: rounded pill, strong text contrast, used sparingly
- Drawer / modal: large radius, restrained width, generous padding
- Bottom action bar: sticky, high contrast primary button, safe-area aware

## Navigation
- Desktop: clean left sidebar or top-plus-side hybrid
- Mobile: simplified; canvassing mode removes distractions
- Top-level: Dashboard, People, Canvassing, Follow-ups, Outreach, Donors, Team, Settings

## Avoid
- Cramped dashboards
- Generic SaaS styling
- Patriotic or cliché political visuals
- Tiny text or tiny buttons
- Unnecessary animation
- Dense forms
- Complex nested panels
